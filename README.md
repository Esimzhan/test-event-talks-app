# BigQuery Release Notes Sentinel (BQ Sentinel)

Веб-приложение на Flask, HTML, CSS и JavaScript, предназначенное для удобного мониторинга официальных обновлений Google Cloud BigQuery и быстрой публикации важных изменений в X (Twitter).

---

## 🌟 Возможности приложения

- **Интеграция с официальным фидом**: Считывает XML-фид обновлений BigQuery в режиме реального времени.
- **Умный парсер контента**: Делит массивные ежедневные логи от Google на отдельные карточки-события на основе тегов `<h3>`.
- **Категоризация обновлений**: Автоматически распределяет события по типам: *Фичи (Features)*, *Критические изменения (Breaking)*, *Проблемы (Issues)* и *Остальное (Others)*.
- **Встроенный кэш**: Хранит данные в оперативной памяти на 10 минут, снижая сетевую задержку и нагрузку на сервера Google.
- **Обновление по требованию**: Кнопка «Refresh» со спиннером позволяет в любой момент принудительно обновить кэш.
- **Интерактивный Tweet Composer**:
  - Генерация краткого описания с хэштегами (`#BigQuery`, `#GoogleCloud`).
  - Проверка лимита в 280 символов.
  - Живой предпросмотр (Live Preview) поста в стиле X (Twitter).
  - Публикация в реальный аккаунт X (через Twitter Web Intent) или локальная симуляция отправки с красочным эффектом конфетти на Canvas.
- **Адаптивный интерфейс (с переключением тем)**: Поддержка темного и светлого оформления с сохранением выбора пользователя.

## 🚀 Быстрый запуск

### 1. Требования
Убедитесь, что у вас установлен Python версии 3.10 или выше.

### 2. Клонирование и переход в директорию
```bash
cd \agy-cli-projects\bigquery-release-notes
```

### 3. Активация виртуального окружения (уже настроено)
* **PowerShell**:
  ```powershell
  .\venv\Scripts\Activate.ps1
  ```
* **CMD (Командная строка)**:
  ```cmd
  .\venv\Scripts\activate.bat
  ```

### 4. Установка зависимостей (уже установлены)
Если вы запускаете на новой машине:
```bash
pip install flask
```

### 5. Запуск сервера
```bash
python app.py
```
После запуска приложение будет доступно по адресу: **[http://127.0.0.1:5000](http://127.0.0.1:5000)**.

## 🔧 API Эндпоинты

### `GET /api/releases`
Возвращает список обновлений в формате JSON.

- **Параметры**:
  - `refresh=true` (необязательный): Сбрасывает кэш и принудительно скачивает свежий XML от Google.
  
- **Пример ответа**:
  ```json
  {
    "success": true,
    "source": "fetched",
    "last_updated": "2026-06-16 23:53:13",
    "releases": [
      {
        "id": "tag:google.com,2016:bigquery-release-notes#June_15_2026",
        "title": "June 15, 2026",
        "updated": "2026-06-15T00:00:00-07:00",
        "link": "https://cloud.google.com/bigquery/docs/release-notes",
        "content": "<h3>Feature</h3>\n<p>Use Gemini Cloud Assist...</p>"
      }
    ]
  }
  ```
